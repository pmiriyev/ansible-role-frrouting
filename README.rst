==============
FRRouting role
==============

Ansible role to install and manage FRRouting. This role
is being maintained by OpenStack-Ansible project, though can be used
as a stand-alone role as well.

Documentation for the project can be found at:
  https://docs.openstack.org/ansible-role-frrouting/latest/

The project source code repository is located at:
  https://opendev.org/openstack/ansible-role-frrouting/

The project home is at:
  https://launchpad.net/openstack-ansible

The bugs is at:
  https://bugs.launchpad.net/openstack-ansible
